# Tutorials

These tutorials walk through how to use Starsim. If you want to explore them interactively, you can run them locally either through the [Quarto extension](https://marketplace.visualstudio.com/items?itemName=quarto.quarto) (or [Positron](https://posit.co/products/ide/positron/)), or by running `convert_notebooks.sh` and then starting a Jupyter session (run `jupyter lab` or `jupyter notebook` in this folder).
